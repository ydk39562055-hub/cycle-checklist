# 경기순환 체크리스트 (로컬 전용)

매월 1회, 17개 경제지표 차트를 직접 보며 68개 문항을 체크해
국면별 점수·확률을 집계하고, 시나리오를 기록·추적하는 도구입니다.
**판단의 자동화가 아니라 "판단 훈련 + 기록"이 목적**입니다.

> ⚠ 이 도구는 **내 컴퓨터에서만** 쓰는 로컬 전용입니다. 웹에 올리거나 공개 배포하지 않습니다.

---

## 매달 이렇게 쓰세요 (클릭 순서)

1. **`1_월간 데이터 업데이트.bat` 더블클릭**
   → 최신 경제데이터를 받아오고 체크리스트 화면을 새로 만듭니다. (1~2분)
   검은 창에 `완료` 가 보이면 끝. 아무 키나 눌러 닫습니다.

2. **`2_체크리스트 열기.bat` 더블클릭**
   → 브라우저에 `http://localhost:8765` 가 자동으로 열립니다.
   (이 검은 창은 체크하는 동안 **켜둔 채로** 두세요. 닫으면 저장 기능이 멈춥니다.)

3. 화면에서 지표 차트를 보며 문항을 체크 → **`이 달 저장`** 버튼.
   다 끝나면 2번 검은 창에서 `Ctrl + C` 로 종료.

---

## 4개 탭

- **월간 체크** : 지표별 차트 + 4문항 체크 → 국면별 원점수·이웃가중 점수·확률 자동 집계.
  내 판단 최다국면과 **OECD 경기선행지수(CLI)** 국면의 일치/불일치를 같이 보여줍니다.
  - 회색 작은 글씨 = **참고 의견**(아래 설명). 체크는 직접 하세요.
  - `참고 의견대로 모두 채우기` 버튼으로 초안을 깔고, 개별 수정 가능.
- **히스토리** : 저장된 달들을 국면 점수 스택바로. CLI와 불일치한 달은 빨갛게 강조(=공부할 지점).
  "내가 뒤집은 판단"(참고의견과 다르게 본 문항) 목록도 모아 봅니다.
- **시나리오** : ①6가지 렌즈 브레인스토밍 → ②인과 체인 → ③핵심변수+시나리오+**조기신호(필수)**.
  확정하면 **수정 불가(박제)** 로 보존됩니다.
- **자산군** : 국면 확률(월간 체크 자동연동) × 국면별 과거수익률 → 자산군 기대수익률.
  국면별 수익률은 `config/asset_returns.json` 의 값을 직접 수정·저장하세요(예시 더미가 들어있음).

## 참고 의견(보조)이란?
각 문항 옆 회색 글씨. 지표의 **12개월 추세 기울기 · 6개월 변화 · 5년 범위 내 백분위 · 고점/저점 근접**을
계산해 `그렇다 / 아니다 / 애매` 3단계와 근거 수치를 보여줍니다.
어디까지나 참고용이고, 최종 체크는 본인이 합니다. 참고와 다르게 체크하면 기록에 `뒤집은 판단`으로 남습니다.

## ISM 제조업 PMI 만 수동
무료 데이터 소스가 없어 차트가 자동으로 안 나옵니다. ISM 보고서/트레이딩뷰 등에서
직접 확인하고 그 지표의 4문항만 체크하세요.

## 폴더 구조
```
config/   questions.json(68문항)·indicator_meta.json(지표설명)·asset_returns.json(수익률표)
data/     series.json(차트데이터)·records/(월별 기록)·scenarios/(박제 시나리오)·cache/(원본 캐시)
          .fred_key  ← FRED API 키(로컬 전용, 절대 공유/커밋 금지)
output/   checklist.html  ← 실제 화면(데이터 임베드된 단일 파일)
parse_questions.py  엑셀→문항 변환 (문항/지표 정의 바뀔 때만 실행)
fetch.py            데이터 수집     build.py  HTML 생성
run_monthly.py      위 둘을 한번에   serve.py  저장 서버   selftest.py  자가검증
```

## 데이터 출처
- FRED 공식 API (실업률·산업생산·금리 등 15개 + OECD CLI) — 키는 `data/.fred_key`
- yfinance (주가 S&P500, 원자재 GSCI)
- ISM PMI는 수동

## 점검하려면
`python selftest.py` → 문항 68개·가중치·이웃가중(6/2/15/9→11.5/12.5/20.5/19.5)·더미 저장까지 자동 검증.
